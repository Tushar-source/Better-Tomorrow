Javascript

  • Javascript is a scripting language.
  • Javascript is implemented by Brendan in 1995 at Netscape.
  • It was initially named as Livescript.
  • Latest version is ES11, released in 2020.
    (ES -> ECMA script)
  • Javascript programs(scripts) can be executed by either web browsers or Node Engines.
  • Javascript is Object Based Scripting Language.
  • Javascript supports predefined objects, Eg
    • window
    • document
    • console, etc

Uses Of Javascipt

  • Scripting
  • Making Page as dynamic
  • Client side validation
  • modern UI framework application
    • AngularJS
    • ReactJS
    • VeuJS
    • NodeJS
    • etc

JS Variables

  • Variables are used to store data.
  • By using variables we can store any type of data,
    Eg numbers, strings, files, audio, video, etc.
  • Variables can be declared using 'var', 'let', 'const'.
  • 'let' and 'const' keywords introduced in ES5.


For Example

                            var  x = 5;
                            let y = 6;
                            const  price1 = 5;
                            const  price2 = 6;
                            let total = price1 + price2;
