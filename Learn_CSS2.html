CSS

  • CSS stands for Cascading Stylesheet.
  • It is a stylesheet language used for describing the presentation of document written in a markup language like HTML.
  • CSS is a cornerstone technology for world wide web alongside HTML and Javascipt.
  • CSS is designed to enable separation of presentation and contents.
  • All styles are stored in CSS files.
  • CSS files must have extension as '.css' only.


WHY CSS?

  • It allows much ritcher presentation appearence.
  • It reduces workload by centrallizing commands for visual appearence instead of scatterd ones.
  • Same styles can be reused for multiple documents.


History

  • First version of css was released in 1996.
  • Second version of css was released in 1998, with improved table fetures.
  • Third version of css was released in 2008 and currently in use with improved support for modules,it also supports modern browsers.
  • Fourth version was released in 24th march 2017,which is faster than css3

Types of CSS

  2. Internal CSS-   Medium priority, can be applied to only that document

  3. External CSS-  Lowest priority, can be applied globally

CSS syntax (rule-set)

  • CSS syntax is selector and declaration block.
  • The selector points to the html element you want to style.
  • Each declaration includes CSS property and value pair.
  • Propertes and values are separated by colon(:)
  • Each property and value separated by semicolon(;)


For Example

                            h1
                            {
                                color : red;
                                font-size : 12px;
                            }
                            where, h1 is a selector color is property red is value 
 font-size is property 12px is value

CSS Selectors

  • CSS selectors are used to select the contents you want to style.
  • Types-
    1. Element selector
    2. Id selector
    3. Class selector
    4. Universal selector
    5. Group selector

1. Element Selector

  • The element selector selects the html element by name.

Syntax:
                            element
                            {
                                style1;
                                style2;
                                ...
                            }

2. Id Selector

  • It uses id attribute of html element.
  • To select a selector with specific id (#) is used.

Syntax:
                            
                            #id
                            {
                                style1;
                                style2;
                                ...
                            }

3. Class Selector

  • The class selector selects html elements with specific class attribute.
  • It is used with a "period" symbol (. DOT) followed by class name.

Syntax:
                            
                            . class_name
                            {
                                style1;
                                style2;
                                ...
                            }

4. Universal Selector

  • The universal selector is used as wildcard selector.
  • It selects all elements in page.
  • It is denoted by '*' symbol

Syntax:
                            
                            *
                            {
                                style1;
                                style2;
                                ...
                            }

5. Group selector

  • Group selector is used to select all elements with same style definition
  • commas are used to separate selectors in grouping.

Syntax:
                            
                            selector1, selector2,selector3....
                            {
                                style1;
                                style2;
                                ...
                            }

CSS Background

The CSS background properties are used to add background effects for elements.

background-color

background-color specifies background color of element

For Example: <h1 style="background-color: rgb(0,0,255);"> I am heading 1</h1>



background-image

background-image adds image to background

Syntax: background-image : url();



background-repeat

background-repeat repeats images either in horizontal direction, or in vertical direction, or no repeat.

background-repeat :

  • repeat-x
  • repeat-y
  • no-repeat



background-position

background-position:

  • top
  • bottom
  • left
  • right

For Example: background-position : right top;



background-attachment

The background-attachment specifies image should be fixed or scrolling

background-attachment:

  • fixed

    - not fixed with page

  • scroll

    - fixed with page



background-size

background-size specifies size of background

background-size :

  • contain;
  • cover;
  • size(number);

CSS Gradient Backgrounds

There are two types of Gradient

  1. Linear Gradient
  2. Radial Gradient
Note:-

minimum 2 color stops required

1. Linear gradient

Syntax
                            background : linear-gradient(direction/degree, color_stop1, color_stop2, ...)

where direction:

  • to top
  • to bottom
  • to left
  • to right bottom
    • ......
degree
  • 90deg
  • 180deg
  • -90deg


2. Radial Gradient

Syntax
                            background : radial-gradient(shape, color_stop1, color_stop2, ...)

where shape:

  • circle
  • ellipse
