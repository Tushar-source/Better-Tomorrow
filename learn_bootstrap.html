Bootstrap

  • Bootstrap is opensource framework of CSS.
  • It helps in desiging responsive and device friendly web pages.(RWD - responsive web design)
  • Bootstrap includes CSS and optional scripting (javascript) templates.
  • This framework helps in faster web development.

History

  • Bootstrap was developed by Mark otto and Jacob Thornton at Twitter in Aug 2011.
  • Bootstrap 2.x was released on 31st Jan 2012, with builtin support for Glyficons.
  • v2.x onwords support for RWD.
  • Bootstrap 3 was released on 19th Aug 2013.
  • It focused on mobile first approach.
  • Bootstrap 4 was announced on 29th october 2014 and stable release was on 18 Jan 2018.
  • Stable version is 4.5
  • Bootstrap 5
    • alpha version : 16th Jun 2020
    • beta 1 : 7th Dec 2020
    • beta 2 : 10th Feb 2021
    • v5 : 5th May 2021
    • v5.1.0 : 4th Aug 2021
    • Latest 5.1.3 : 9th October 2021
  • Official Website:- https://getbootstrap.com/

Bootstrap Colors

Bootstrap supports following colors

  • primary
  • secondary
  • success
  • danger
  • warning
  • info
  • dark
  • light
  • white
  • black
  • white-50
  • black-50

For Example

                            <p class="h1 text-primary bg-warning">This is Primary Color</p>        
                            <p class="h1 text-secondary bg-danger">This is Secondary Color</p>
                            <p class="h1 text-success bg-info">This is Success color</p>
                            <p class="h1 text-warning bg-dark">This is Warning Color</p>
                            <p class="h1 text-danger bg-light">This is Danger color</p>
                            <p class="h1 text-info bg-white">This is Info color</p>
                            <p class="h1 text-dark bg-black">This is dark color</p>
                            <p class="h1 text-light bg-white-50">This is light color</p>
                            <p class="h1 text-white bg-dark-50">This is white color</p>
                    
                            <p class="h1 text-black bg-primary">This is black color</p>
                            <p class="h1 text-white-50 bg-secondary">This is 50% white color</p>
                            <p class="h1 text-black-50 bg-success">This is 50% black color</p>

Note:

Mark text as important:- use class ".lead"

Mark text as Unimportant:- use class ".muted"

Bootstrap Lists

List

  • list-group

  • list-group-item-*...* -> colors
  • list-group-horizontal
  • active

To create a basic list group, use an <ul> element with class '.list-group', and <li> elements with class '.list-group-item'
